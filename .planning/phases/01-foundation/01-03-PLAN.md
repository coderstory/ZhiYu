<objective>
Create the complete MIUI-style theme system for the ZhiYu app: color palette with dark mode, MIUIX theming, CJK font fallback, shape tokens, and spacing scale.

**Purpose:** Establishes the visual foundation that every downstream phase (Navigation Shell, Info Dashboard, Knowledge Base, Profile/Settings) uses. The `ZhiYuTheme` composable wraps both MIUIX and Material3 theme layers. CJK font fallback via `Typeface.CustomFallbackBuilder` ensures Chinese text never renders as tofu.

**Output:** Color.kt (MIUI palette + Material3 color schemes), Type.kt (CJK font fallback bridge + MIUIX text styles + Material3 typography), Shape.kt (MIUI rounded corners), Dimens.kt (8-point spacing grid), Theme.kt (ZhiYuTheme wrapper), CJK font resource, splash and launcher icon drawables.

**Theme architecture (per UI-SPEC.md):** ZhiYuTheme nests MaterialTheme outside MiuixTheme. MIUIX uses CompositionLocalProvider directly (does NOT wrap MaterialTheme). Material3 components (Scaffold, NavigationBar) require MaterialTheme context. Nesting MaterialTheme outside MiuixTheme ensures both component sets see correct theme values.
</objective>
